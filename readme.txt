A plugin for contractors cloud users to create a form that will submit to the contractors cloud app.

== Description ==
The official Contractors Cloud contact form plugin. This plugin generates a contact form with customized text and fields. When a person fills out the contact form, their info will get inserted into the Contractors Cloud app.

== Installation ==
1. Install the plugin.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Configure the plugin via the \"CC Form API\" plugin page found on the main menu.
4. Paste the following shortcode on any page where you want the form: **[cc_contact_form]**

== Frequently Asked Questions ==
= Where do I get an API Key? =
An API Key is generated in the Contractors Cloud app, on the \"API Keys\" page

= Have other questions? =
Read the plugin documentation found on the Wordpress Plugin page in the Contractors Cloud app.
